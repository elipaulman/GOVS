Instructions

Follow these steps to use the Attendance Processor:

  1. Generate your weekly attendance report on VLA and save it as a CSV file.
  2. Upload the CSV file as your weekly attendance report.
  3. Save the emailed attendance report for all students as a CSV file.
  4. Upload the CSV file as the regular attendance report.

Note: The generated report will only include students whose names match between the VLA weekly attendance report and the full list of students. As a result, the report will only show your students.
