Instructions

Follow these steps to use the Attendance Processor:

  1. Generate your weekly attendance report from VLA and save it as a CSV file.
  2. Download the emailed attendance report for all students as a CSV file.
  3. Upload both files using the upload interface on the main page.
  4. Select the desired sort order (e.g., sorting by required hours will organize students by their start date).
  5. Choose the columns to include in the final report.
  6. Reorder the output columns, if needed, to match your preferences.
  7. Click the "Generate" button to create a consolidated attendance report.
Note: The generated report will only include students whose names match between the VLA weekly attendance report and the full list of students.
